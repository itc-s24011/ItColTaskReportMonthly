# UI 参考画像の配置場所

このディレクトリには、外部設計書で使用するUI参考画像を配置します。

## 📸 必要な画像

以下の2つの画像を配置してください：

### 1. タスク登録・実行画面
**ファイル名**: `ui_task_entry.png`

**推奨サイズ**: 横 375px 以上（モバイル画面サイズ）

**含まれる要素**:
- カレンダー表示（例：2020年4月5日）
- 当日の作業時間表示（例：0.0h）
- プロジェクト名入力フィールド
- タグ入力フィールド（会議、メールなど）
- メモ入力エリア
- STARTボタン（タイマー開始用の大きな円形ボタン）
- 編集・削除ボタン

### 2. 月次集計・レポート画面
**ファイル名**: `ui_monthly_report.png`

**推奨サイズ**: 横 375px 以上（モバイル画面サイズ）

**含まれる要素**:
- 年月選択（例：2020年1月）
- 総日数・総時間表示（例：31 DAYS / 132.0 HOURS）
- プロジェクト別一覧表示
  - プロジェクト名（一部マスキング可）
  - 各項目の作業時間（例：21.8h）
  - 割合表示（例：16.5%）
- 横棒グラフによる可視化
- 下部のタブ切り替え（カレンダー、プロジェクト、タグ、設定）

## 📋 画像形式

- **推奨形式**: PNG（透過背景不要）
- **代替形式**: JPG も可
- **ファイルサイズ**: 500KB 以下を推奨

## 🔒 プライバシー

実際のプロジェクト名や個人情報が含まれる場合は、適宜マスキング（黒塗り・モザイク）してください。

## ✅ 配置後の確認

画像を配置したら、以下のコマンドで外部設計書を開いて表示を確認してください：

```bash
# ブラウザで外部設計書を開く
open ../external_design.html
# または
firefox ../external_design.html
```

画像が正しく表示されていれば完了です！

---

**注**: 画像が配置されていない場合、外部設計書にはプレースホルダーが表示されます。開発の参考として、実際のUI画像を配置することを強く推奨します。
